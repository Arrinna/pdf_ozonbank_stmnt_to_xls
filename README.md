# PDF Ozon Bank Statement to Excel

Скрипт для извлечения операций из PDF-выписки Ozon Bank и сохранения их в Excel.

На входе берется PDF-файл выписки, лежащий в корне проекта. При обычном запуске скрипт автоматически выбирает самый новый PDF по дате создания, извлекает операции и сохраняет результат в `.xlsx`.

## Что делает скрипт

- открывает PDF-выписку через `pdfplumber`;
- пытается извлечь таблицы со всех страниц;
- удаляет строки, похожие на заголовки;
- склеивает многострочные записи;
- выделяет дату, номер документа, назначение и сумму операции;
- сохраняет результат в Excel.

Итоговая таблица содержит столбцы:

- `date`
- `document`
- `purpose`
- `amount_rub`

## Требования

- Python `3.11+`
- `uv` для установки зависимостей и запуска в виртуальном окружении

## Установка

```powershell
uv sync
```

Эта команда создаст виртуальное окружение `.venv` и установит зависимости из `pyproject.toml` и `uv.lock`.

## Запуск

```powershell
uv run python parse_pdf_statement.py
```

Скрипт:

- найдет в корне проекта самый новый PDF-файл;
- извлечет из него операции;
- создаст Excel-файл в корне проекта.

Если имя PDF содержит хвост вида `_26859557.pdf`, этот идентификатор будет использован в имени результата:

```text
ozonbank_transactions_only_26859557.xlsx
```

Иначе будет создан файл:

```text
ozonbank_transactions_only.xlsx
```

## Как это работает

Основная логика находится в [parse_pdf_statement.py](/c:/Users/USER/Projects/pdf_ozonbank_stmnt_to_xls/parse_pdf_statement.py):

- `extract_statement_id()` извлекает ID выписки из имени файла;
- `parse_rub_amount()` преобразует строку суммы в число;
- `parse_ru_dt_from_first_cell()` извлекает дату и время;
- `merge_multiline_rows()` склеивает переносы строк в одной операции;
- `extract_transactions()` собирает итоговый `DataFrame`;
- `pdf_to_excel_transactions_only()` сохраняет данные в Excel.

## Ограничения

- Скрипт рассчитан на текущий формат PDF-выписок Ozon Bank.
- Если банк изменит верстку таблицы или порядок колонок, парсер может потребовать доработки.
- Выбор входного файла сейчас идет только из корня проекта.
